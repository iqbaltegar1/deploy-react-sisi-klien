# 📖 Deskripsi Project

Project ini merupakan aplikasi web berbasis **React + Vite** yang dikembangkan sebagai tugas mata kuliah **Pemrograman Sisi Klien**.

Aplikasi ini menerapkan konsep **Single Page Application (SPA)** menggunakan React serta telah berhasil dideploy menggunakan **Vercel** sehingga dapat diakses secara online.

# ⚙️ Cara Menjalankan Project

## 1. Clone Repository

```bash
git clone https://github.com/iqbaltegar1/deploy-react-sisi-klien.git
```

## 2. Masuk ke Folder Project

```bash
cd deploy-react-sisi-klien
```

## 3. Install Dependency

```bash
npm install
```

## 4. Jalankan Development Server

```bash
npm run dev
```

Project akan berjalan pada

```
http://localhost:5173
```

---

# 📦 Build Project

Untuk membuat versi production:

```bash
npm run build
```

Untuk melakukan preview hasil build:

```bash
npm run preview
```
